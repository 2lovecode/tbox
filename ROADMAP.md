# 产品规划文档 v2.0

基于国内外主流在线工具网站（toolhelper.cn、sojson.com、lddgo.net、jyshare.com）功能分析。

## 产品定位

**目标用户**: 后端/测试/运维工程师
**核心价值**: 本地化、安全、高效的百宝工具箱
**规划工具**: 150+

## 核心发现

- **高频工具**: JSON/XML处理、加密解密、编码转换、正则测试
- **特色功能**: 国密算法（SM2/3/4）、坐标转换、图片处理
- **用户体验**: 实时预览、批量处理、历史记录

## 工具分类与优先级

### 🔥 P0 - 立即开发（30个）

**JSON/XML/YAML处理**
- JSON转实体类、JSON对比、XML/YAML格式化、JSON转YAML

**加密安全**
- JWT、AES、RSA、SM2/3/4、HMAC、Base16/32/58/62、URL编码、Unicode转换

**编码字符**
- 进制转换、IEEE 754、大小端转换、ASCII/Unicode查询

**文本处理**
- 正则测试、文本对比、去重排序、驼峰转换、批量替换

**时间日期**
- 时间戳转换、Cron表达式、日期计算

**网络工具**
- HTTP请求、DNS查询、端口检测、SSL证书、IP计算、端口占用

**数据库**
- SQL格式化、连接测试、建表转换、数据脱敏

**图片处理**
- 格式转换、裁剪、Base64互转、颜色拾取

**数据转换**
- CSV/JSON/Excel互转、存储单位转换、随机数/密码生成、UUID

**日志监控**
- 日志分析、格式化、正则提取

**部署运维**
- Cron、Docker、Nginx、chmod计算、curl转代码

### ⭐ P1 - 近期开发（50个）

XPath/JSON Path测试、DES加密、字符集转换、Gzip压缩、HTML实体、文本分割合并、日期计算、Whois查询、SSL生成、Ping/TraceRoute、CSV转SQL、JSON转SQL、调色板、日志统计、进程管理、文件查找、二维码生成、秒表倒计时、随机分组、单位换算。

### ⚪ P2 - 中期开发（70个）

TOML/INI处理、密码强度检测、Markdown编辑器、DNS传播、网络延迟监控、ER图生成、SQL优化、图片编辑器、SVG编辑、思维导图、配置对比、环境变量、公网IP查询、键盘测试、摄像头测试等。

## 特色功能

- **工作流系统** - 多工具串联，自定义流程
- **批量处理** - 批量文件/数据操作
- **历史记录** - 操作历史，快速恢复
- **收藏夹** - 收藏常用工具
- **快捷键** - 全局快捷键唤起
- **云端同步** - 配置跨设备同步（可选）
- **主题定制** - 深色/浅色模式

## 技术实现

**前端**: Vue 3 + TypeScript + Vite + Monaco Editor + Diff2Html + Mermaid

**后端**: serde + aes/rsa/sha + libsm + reqwest + rusqlite + image

**优化**: 异步处理 + 流式处理 + 多线程并行

## 竞争优势

- **本地化** - 数据不上传，零隐私泄露
- **高性能** - Rust后端，速度快
- **全面性** - 150+工具，覆盖全面
- **离线可用** - 核心功能无需网络
- **批量处理** - 支持大批量操作
- **国密支持** - 支持国密算法
